# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-08

### Added
- Initial release of portfolio website
- Hero section with parallax particle background
- About Me section with academic timeline
- Projects showcase with 6 featured projects
- Skills & Services section with animated cards
- Security compliance highlights
- Blog section with category filters
- Contact section with integrated map
- Smooth parallax animations throughout
- Mobile-responsive design
- Glass morphism UI cards
- Floating code snippets animation
- Typing effect for hero subtitle
- Scroll-to-top button
- Smooth scroll navigation
- Custom cursor glow effect (desktop)
- AOS (Animate On Scroll) integration
- Particles.js integration
- Social media links
- Contact form with validation
- Footer with animated elements

### Features
- **Dark Theme**: Deep green (#123513) with neon accents
- **Parallax Layers**: Multi-depth scrolling effects
- **Interactive Cards**: 3D tilt on hover
- **Blog Filters**: Dynamic category filtering
- **Responsive Navigation**: Mobile hamburger menu
- **SEO Optimized**: Meta tags and semantic HTML
- **Performance**: Optimized animations and lazy loading
- **Accessibility**: ARIA labels and keyboard navigation

### Technical Stack
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript (ES6+)
- Particles.js v2.0.0
- AOS v2.3.1
- Font Awesome v6.4.0
- Google Fonts (Inter, JetBrains Mono)

### Documentation
- README.md with comprehensive setup guide
- ASSETS_GUIDE.md for image setup
- DEPLOYMENT.md for hosting instructions
- setup-placeholders.html for quick asset generation
