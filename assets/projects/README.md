- **Dimensions**: 1600x900px (16:9 aspect ratio)
- **Size**: < 500KB per image (optimized)
- **Quality**: High resolution screenshots or mockups

## Tips for Good Project Images

1. **Screenshots**: Use actual project screenshots when possible
2. **Mockups**: Create mockups in Figma/Adobe XD
3. **Diagrams**: Architecture diagrams and flowcharts work great
4. **Dashboards**: Analytics and data visualization screens
5. **UI Elements**: Clean, modern interface screenshots

## Tools for Creating Images

- **Figma** - Free design tool
- **Canva** - Quick mockups
- **Carbon** - Beautiful code screenshots
- **Excalidraw** - Diagrams and architecture
